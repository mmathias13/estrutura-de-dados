Implemente a função vector_push_back: Primeiro, verifique se a capacidade máxima do vetor foi atingida, isto é, se o tamanho é maior ou igual ao espaço alocado. Se sim, utilize a função realloc para aumentar a capacidade máxima para o dobro. Lembre-se de atualizar a variável que armazena o tamanho alocado. Por fim, adicione o novo item na primeira posição livre no vetor. Use o valgrind para verificar se mesmo com os reallocs a memória foi utilização corretamente. O programa main.c (ver arquivo anexo) lê um número n e, em seguida, lê uma sequência de n números. Os números são inseridos em um vector usando a função vector_push_back. Por fim, a função exibe os elementos na tela.
